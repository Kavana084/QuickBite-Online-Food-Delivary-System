# QuickBite 🍔

QuickBite is an online food ordering web application developed using Java technologies.  
The project allows users to browse restaurants, view menus, place orders, and manage food delivery operations efficiently.

## 🚀 Features
- User registration and login
- Browse restaurants and food items
- Add items to cart and place orders
- Order management system
- Responsive and user-friendly interface

## 🛠 Technologies Used
- Java
- JDBC
- Servlets & JSP
- HTML
- CSS
- MySQL
- Apache Tomcat
